# Project Rules & Customizations

## Code File Limits
- **File Length Constraint:** No code files (including CSS, JS, JSX, and Python files) should exceed **200 lines**.
- **Refactoring & Modularization:** If a file grows near or beyond 200 lines, immediately partition and delegate sub-components or helper functions into dedicated modular utility files (e.g. `chartIndicators.js`, `ChartSubComponents.jsx`, `modal.css`).
